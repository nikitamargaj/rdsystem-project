# RD User Management System

A Full Stack CRUD Web Application built using:

- React (Frontend)
- Spring Boot (Backend)
- MySQL (Database)

---

## Features

- Add new user
- Update user details
- Delete user
- View all users
- Search users
- Pagination (if added)

---

##  Tech Stack

Frontend:
- React
- Axios
- Bootstrap

Backend:
- Spring Boot
- Spring Web
- Spring Data JPA

Database:
- MySQL

---

##  How to Run

### Backend


mvn spring-boot:run
### Frontend

npm install
npm run dev
